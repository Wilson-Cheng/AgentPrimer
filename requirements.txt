# Python dependencies for the AgentPrimer embedding sidecar
# ---------------------------------------------------------------------------
# The Knowledge Base (RAG) feature uses scripts/embed_server.py — a small
# stdlib HTTP server that loads a sentence-embedding model and exposes
# POST /embed for lib/rag.ts to call.
#
# Without these packages installed, the sidecar still starts but runs in
# "degraded" mode: /embed returns HTTP 503 and lib/rag.ts falls back to its
# FTS5 keyword search. The rest of AgentPrimer (chat, agents, tools, MCP)
# is fully functional without Python — so installing these is OPTIONAL,
# but recommended if you want semantic search over uploaded documents.
#
# Install with:  pip install -r requirements.txt
#
# Pinned to known-good major versions only; let pip resolve the minor/patch.

# Primary backend — ONNX-based, no PyTorch, ~200 MB.
# Works on Alpine via the gcompat package, on Debian, macOS, and Windows.
fastembed>=0.4,<1.0

# Fallback backend if fastembed isn't available on your platform.
# Pulls PyTorch (~2 GB) so prefer fastembed where it works.
# sentence-transformers>=3.0,<4.0
