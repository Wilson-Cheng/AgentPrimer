# Maintainers

AgentPrimer is currently maintained by:

- Wilson Cheng — project owner and primary maintainer

## Maintainer responsibilities

- Review and merge pull requests.
- Triage bugs, feature requests, and security reports.
- Keep release notes and documentation aligned with shipped behavior.
- Coordinate security disclosures according to `SECURITY.md`.

## Becoming a maintainer

Regular contributors may be invited as maintainers after sustained, high-quality contributions and demonstrated care for the project's educational and security goals.
