# Support

## Getting help

- Use GitHub Issues for reproducible bugs and feature requests.
- Use the documentation in `README.md` and `docs/` for setup, architecture, and course material.
- Include your AgentPrimer version or commit hash, Node.js version, deployment method, and relevant logs when requesting help.

## Security reports

Do not open public issues for vulnerabilities. Follow `SECURITY.md` for private reporting.

## Scope

AgentPrimer is a self-hosted educational project. Community support is best-effort unless a separate support agreement exists.
